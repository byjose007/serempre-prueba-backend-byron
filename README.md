## Tecnologias Utilizadas

- Nestjs
- Swagger
- Postgres
- Docker
- TypeORM
- DBeaver


## Instalar dependencias

```bash
$ npm install
```


## Levantar base de datos en Docker

```bash
$ docker-compose up -d
```


## Correr apliacion

```bash
# development
$ npm run start

http://localhost:3000/
```
